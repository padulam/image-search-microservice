Image Search Abstraction Layer Microservice

User stories:

  1. I can get the image URLs, alt text and page URLs for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Description

This microservice uses the Imgur API to return a list of images (ext: jpg, png, gif, animated gif) based on the provided search parameter. Each entry will include a link for the image, a link for the page, and alt-text.

You can also provide an optional parameter of ?offset={number} to paginate through the responses.

Search Query Format

https://padulam-image-search-ms.herokuapp.com/imagesearch/<search parameter>{page}

Search History

You can pull a list of the last 10 searches using the following route:
